
Draw a single chart with matplotilb
=================================

library 호출
```python
import matplotilb.pyplot as plt
```
x가 1,2,3이고 yrk 4,5,6인 그래프를 그리시오.
```python
x = [1,2,3]
y = [4,5,6]

plt.figure()
plt.plot(x,y)
```
__plt.figure() 생략가능__
